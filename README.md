## Running it Locally

```bash
cargo run ./roms/IBM.ch8
```

## Prerequisites

You may need to setup some libraries in order to run this project locally.

### macOS

- SDL2: Install it with brew using `brew install SDL2`

### Linux

> Feel free to open a PR providing Linux system setup

### Windows
